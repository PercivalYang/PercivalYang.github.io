## 数据隐藏

- **数据抽象**：仅向用户暴露接口，隐藏具体的实现细节，如：

```c++
#include <iostream>
using namespace std;
 
class Adder{
   public:
      // 构造函数
      Adder(int i = 0)
      {
        total = i;
      }
      // 对外的接口
      void addNum(int number)
      {
          total += number;
      }
      // 对外的接口
      int getTotal()
      {
          return total;
      };
   private:
      // 对外隐藏的数据
      int total;
};
```

- **数据封装**：把数据和操作数据的函数捆绑在一起，如上述程序块
  - 把一个类定义为另一个类的**友元类**会暴露实现细节，降低封装性。
